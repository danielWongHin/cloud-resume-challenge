# cloud-resume-challenge

This Project is going to demostrate a demo resume page hosting in Azure. In Azure, the serverless approach is decided to use by **Azure Function** and **CosmosDB**. Since the cost for the Azure entry-point service is pricier than I thought, we have decided to use **Cloudflare** as replacement.
